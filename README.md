# 🖨️ Lukas' 3D Models
 
A personal collection of 3D models designed in Fusion 360 and printed on a Prusa printer. All files are provided in `.3mf` format, ready to slice in PrusaSlicer.
 
---
 
## 📦 Categories
 
### 🏠 Home & Practical
Everyday items designed to solve real problems — holders, clips, mounts, door stops, and organizers.
 
### 🎲 Board Game Inserts
Custom inserts for board games, including **Eldritch Horror** and others. Designed to keep components sorted and the box tidy.
 
### 🔧 Gadgets & Fun Stuff
Miscellaneous prints — useful gadgets, fun objects, and experimental designs.
 
---
 
## 📁 File Format
 
All models are provided as `.3mf` files (PrusaSlicer project format), which includes:
- Model geometry
- Recommended print settings where applicable
 
---
 
## 🖨️ Recommended Print Settings
 
| Setting | Value |
|---|---|
| Layer height | 0.2 mm |
| Infill | 15–20% (varies by model) |
| Supports | See individual model notes |
| Filament | PLA (unless otherwise noted) |
 
> Some models may include specific notes about orientation, supports, or filament type.
